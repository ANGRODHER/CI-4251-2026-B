# CI-4251

## Programa del curso

1. Introducción: repaso de Haskell, funciones recursivas, Monoids, Funtores & Foldable.

2. Funtores, Aplicativos, Monadas concretas:
    - Maybe
    - Either
    - Identity
    - List
    - Reader
    - Writer
    - State
    - Logic
    - Cont
    - Compose
    - Free
    - Parser

3. Monad Transformers (MTL) y composición de funtores/aplicativos

4. Parser combinators (Parsec y metodo clásico)

5. Optics: The Lens Library (getters, setters y otras generalizaciones funcionales)

6. Concurrencia, Paralelismo, STM y Estrategias de Evaluación

7. QuickCheck & HSpec: property testing

8. Effect Systems: MTL vs Polysemy ft: Bluefin.

Posible pero no seguro:

- Morfismos (recursion schemes)
- Zippers
- Profiling
- Functional Graphs (fgl)
- Arrows & Functional Reactive Programming (FRP)
- DB: Opaleye, Esqueleto
- DSL: Deep/Shallow embedding.
- GADTs, Type Families, DataKinds, Type-level programming.
- Metaprogramming: Template Haskell.
- Rank-N-Types, Existential Types.
- Monomorphism Restriction
- Functional Dependencies
- Linear Types
- Quantified Constraints
