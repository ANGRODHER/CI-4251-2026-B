# CI4251


## Programa del curso

1. Introduccion: repaso de haskell, funciones recursivas, Monoids, Functors & Foldable.
2. Functors, Aplicatives, Monads concretos:
    - Maybe
    - Either
    - Identity
    - List
    - Reader
    - Writer
    - State
    - Logic
    - Cont
    - Compose
    - Free
    - Parser
3. Monad Transformers (MTL) y composicion de functores/aplicativos
4. Parser combinators (Parsec y metodo clasico)
5. Optics: The Lens Library (getters, setters y otras generalizaciones funcionales)
6. Concurrencia, Paralelismo, STM y Estrategias de Evaluacion
7. QuickCheck & HSpec: Property testing
8. Effect Systems: MTL vs Polysemy ft: Bluefin.
